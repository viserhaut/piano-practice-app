Bluetooth MIDI 接続ガイド

KAWAI CX302 → MacBook Air（Bluetooth MIDI）

  1. CX302 の Bluetooth を有効にする
    デフォルトは OFF です。METRONOME + SOUND SELECT を同時に5秒長押しで ON になります。
  2. Mac の「Audio MIDI設定」を開く
    Spotlight（⌘+スペース）→「Audio MIDI 設定」と入力 → Enter
  3. MIDI スタジオを表示する
    メニューバー「ウィンドウ」→「MIDIスタジオを表示」
  4. Bluetooth 構成を開いて CX302 を「接続」
    MIDIスタジオ上部の「Bluetooth」ボタンをクリック
    → CX302 が表示されたら「接続」をクリック
  5. このページで「MIDI接続」をクリック
    ブラウザの許可ダイアログで「許可」を選択
    → 自動的に「CX302 Bluetooth」と表示されれば完了！
うまくいかない場合
  • CX302 が Bluetooth 構成に出ない → 手順①のボタン長押しを再試行
  • 「デバイスが見つかりません」→ 手順④が完了しているか確認してから「MIDI接続」を再クリック
  • Chrome または Edge を使用しているか確認（Safari は Web MIDI 非対応）
  • USB ケーブル（USB-B → USB-A）でも接続できます（最も確実）
